## Project overview
This project simulates all LAB requirements from 1 to 5. Requirement for each LAB:
- LAB 1 : Simple animtion with LEDs.
- LAB 2 : Timer Interrupt and LED Scanning.
- LAB 3 : Button and switches
- LAB 4 : A cooperative scheduler
- LAB 5 : Flow and Error Control in Communication
All details, instructions and report templates are in the ReportLabs section.
The source code for each LAB is in the SrcSTM32 section.
